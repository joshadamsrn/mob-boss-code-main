# Game Loop

Gameplay is continuous with phases driven by murders and trials.

## Phase 1: Information Seeking
- Players interact, trade, and gather information.
- Phase ends when a murder occurs.

## Phase 2: Trial
- Triggered by death report.
- No purchases or murders during trials.
- Trial ends with jury verdict.

## Phase 3: Reset
- Game returns to Phase 1.
- New round begins.
