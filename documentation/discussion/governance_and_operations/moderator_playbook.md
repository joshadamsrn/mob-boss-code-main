# Moderator Playbook

Responsibilities:
- Confirm deaths (honor system validation).
- Enforce no actions during trial.
- Handle disputes and edge cases.
- Manage physical stashes (if used).
- Pause, resume, end, and start games.
- Create and launch game rooms.
- Remove players before launch.
- Silence ghost players or disable ghost view if needed.
- Mark players dead at any time.
- Moderator is final authority on unresolved disputes.
